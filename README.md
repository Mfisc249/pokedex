# Pokédex

This is a school project.

The purpose of this project is to practice working with an external API and to improve the understanding of JavaScript.  
Data is fetched from the PokéAPI and rendered dynamically in the browser.

The main focus is on:
- Working with fetch and asynchronous JavaScript
- Understanding API structures and data handling
- Dynamic DOM manipulation
- Basic responsiveness and user interaction

This project is intended for learning purposes only.
